# Python学生就业管理系统 python201

## 项目概述

本项目是基于Python语言及Django框架开发的一款学生就业管理系统，涵盖了学院、专业、企业、岗位、管理员、学生等多方面的信息管理，致力于为学校和学生提供一个便捷、高效的就业信息管理平台。

## 技术栈

- **后端语言与框架**：Python 3.0以上，Django
- **前端技术**：HTML、CSS、JavaScript、jQuery
- **数据库**：MySQL 8.0
- **开发工具**：PyCharm、Navicat

## 功能模块

- 学院信息管理
- 专业信息管理
- 企业信息管理
- 岗位信息管理
- 管理员信息管理
- 学生信息管理
- 学生教育经历管理
- 学生项目经历管理
- 学生投递记录管理
- 修改个人信息与密码

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- Python 3.0以上
- MySQL 8.0
- 相关依赖库与工具：请参照文档中的详细安装指导
